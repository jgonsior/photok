Message: [[message]]

The message above is set in the Angular file static/main.js 

var photokApp = angular.module('photok', []);
This line creates the Angular application. 
photokApp.config(['$interpolateProvider', function($interpolateProvider) {
        $interpolateProvider.startSymbol('[[');
        $interpolateProvider.endSymbol(']]');
      }]);
We do that to avoid conflicts with Flask (Jinga templates).

What are the different files ?

  • static/main.jsAngular javascript file
  • templates/HTML templates (Jinga)
  • templates/basic_page.htmlBasic template (parent)
  • templates/pages/Different pages (inherit from basic_page.html)
  • templates/content/Specific content for each page
  • templates/generic/Content that is the same for each page
